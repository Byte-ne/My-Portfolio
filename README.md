# Tanay Mishra — Portfolio

Contact: kilobyte136@gmail.com · GitHub: https://github.com/Byte-ne

## About
This is my personal portfolio showcasing projects, skills, achievements and studies. It’s built as a lightweight static site (HTML, CSS, JS).

## Features
- Clean dark theme with animations
- Responsive layout for mobile and desktop
- Skills progress bars and achievements
- Interactive studies chart (canvas)
- Contact links: Email and GitHub

## Run locally
1. Clone the repo:
   git clone https://github.com/Byte-ne/My-Portfolio.git
2. Open the project folder and serve or open index.html:
   - Quick: open `index.html` in a browser
   - Or run a simple server:
     `python -m http.server 8000` then visit `http://localhost:8000`

## Customize
Edit `data.js` to update name, contact, skills, projects and studies.

## Deploy
Enable GitHub Pages for the repository (Settings → Pages) to host the portfolio.
